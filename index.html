Инструкция по установке VPN

Следуйте этим шагам, чтобы настроить VPN на вашем устройстве.

1. Скачайте конфигурацию

Скачайте файл конфигурации: client.ovpn.

2. Установите OpenVPN

Скачайте и установите приложение OpenVPN:

  • OpenVPN для Windows/macOS
  • OpenVPN для Android
  • OpenVPN для iOS

3. Импортируйте конфигурацию

Откройте приложение OpenVPN и импортируйте скачанный файл client.ovpn.

4. Подключитесь к VPN

Нажмите кнопку подключения в приложении OpenVPN.

Готово!

Теперь ваш интернет-трафик защищён через VPN.
